To create your widget follow the official Microsoft documentation here and here, but it essentially boils down to the following steps:

  1. Replace the default html, css, images and javascript in this project with your own (can include typescript and javascript frameworks) and test locally just like any other html page
  2. Edit the manifest file (vss-extension.json) to suit your extension as per the official documentation
  3. Package and publish (privately or publicly) your extension as per the official documentation
